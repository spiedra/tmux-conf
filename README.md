My own tmux configurations.

## Getting Started

Prerequisites

- Tpm: https://github.com/tmux-plugins/tpm

Installation

1. Clone tpm

```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

2. Install plugins

```bash
Ctrl + I
```
